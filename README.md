# Exploratory Data Analysis and Statistical Analysis Notebooks

This repository contains a set of Jupyter Notebooks for Exploratory Data Analysis (EDA) and Statistical Analysis on various datasets. All notebooks are designed to run in Google Colab.

## Notebooks

1. **Data Loading and Initial Exploration**  
   - Description: Demonstrates how to load datasets into Colab and perform basic checks, including data types, missing values, and summary statistics.

2. **Univariate Analysis**  
   - Description: Focuses on single-variable analysis using descriptive statistics, distribution plots, and simple outlier detection.

3. **Notebook 3: Bivariate Analysis**  
   - Description: Explores relationships between two variables using scatter plots, correlation analysis, and basic hypothesis testing.

4. **Multivariate Analysis**  
   - Description: Applies multivariate techniques such as dimensionality reduction, clustering, and advanced visualization methods.

5. **Statistical Modeling**  
   - Description: Introduces core modeling approaches, including regression, classification, and time series analysis.

## Usage

To use these notebooks, open the corresponding links in Google Colab. You can then run the cells interactively to analyze your own datasets or explore the provided examples.

## Requirements

- Common Python libraries such as:
  - `pandas`
  - `numpy`
  - `matplotlib`
  - `seaborn`
  - `scikit-learn`  
- Any additional libraries mentioned within individual notebooks
